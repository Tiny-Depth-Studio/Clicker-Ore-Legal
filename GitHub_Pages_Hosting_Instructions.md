# GitHub Pages ile Basit Hosting (Adım Adım)

Aşağıda, oluşturduğum dosyaları (Gizlilik Politikası, Kullanım Şartları, index) GitHub Pages üzerinde barındırmak için adımlar bulunmaktadır.

1. GitHub hesabı oluşturun (https://github.com) veya giriş yapın.
2. Yeni bir repository oluşturun. Örnek isim: `clicker-ore-legal`.
   - Repository'yi **Public** yapabilirsiniz (Privacy Policy açık olması gerekir).
3. Oluşturduğunuz repository'ye dosyaları yükleyin:
   - `Clicker_Ore_Privacy_Policy.html`
   - `Clicker_Ore_Privacy_Policy.md`
   - `Terms_of_Service.md`
   - `index.html`
4. Repository ayarlarına (Settings) gidin -> "Pages" (eski arayüzlerde "GitHub Pages") bölümünü açın.
5. "Branch" seçeneğini `main` (veya kullandığınız branch) yapın ve `/ (root)` seçin.
6. Save / Deploy yapın. Birkaç dakika içinde `https://<kullaniciadi>.github.io/<repo-adi>/` adresinde sayfanız yayınlanır.
7. Yayınlanan URL örneği: `https://recepyazici.github.io/clicker-ore-legal/Clicker_Ore_Privacy_Policy.html`
8. Bu URL'yi Steamworks panelinde ilgili alana ekleyin (aşağıdaki adımlara bakınız).

> Not: Eğer repo ismini `username.github.io` şeklinde yaparsanız doğrudan `https://username.github.io/` altında barındırabilirsiniz.

## Steamworks'e Link Ekleme
1. Steamworks'e giriş yapın.
2. App'ınızı seçin.
3. Sol menüden **Edit Store Page** veya **Store Presence** -> **Legal** ya da **Support Info / Legal Documentation** bölümüne gidin.
4. "Privacy Policy URL" veya benzeri alana GitHub Pages'de yayınlanan **Clicker_Ore_Privacy_Policy.html** linkini yapıştırın.
5. Değişiklikleri kaydedin.
